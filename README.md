# cnode

> 使用 vue.js 搭建的一个高仿 CNODE 社区

### 项目基本介绍
所有内容都是根据 CNODE 社区提供的 [API](https://cnodejs.org/api) 接口实现的。

主要部分为：
  1. 列表的实现
  2. 详情页：
      - 文章详情页
      - 侧边栏：用户个人信息侧边栏
  3. 回复列表
  4. 分页

### 项目基本结构

**组件**：
1. Header 头部
2. PostList 列表
3. Article 文章详情页
4. SlideBar 侧边栏
5. UserInfo 用户个人信息
6. Pagination 分页组件
